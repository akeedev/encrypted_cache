# encrypted-cache

A standalone Python library providing password-based encrypted local caching
with TTL-aware cache-or-compute pattern.

Development usually starts in notebooks/, then is carried on
and sometimes moved into the library. Once it is moved
into a library, a notebook demonstrating the library is added.

Notebooks are usually by me and should not be edited by AI unless explicitly
requested.

# specs
After writing the notebook, specs may be added as a basis for AI generation of
tests and documentation. These will have the format spec/nbfeat-NNN-name.md.

Specs named `rfeat-NNN-name.md` are features reverse-engineered from
existing code rather than written upfront.
